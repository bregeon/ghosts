`ghosts.beam_configs`
---------------------
Beam configurations are defined as simple dictionaries with minimal information.

.. code-block::

    BEAM_CONFIG_0 = {'x_offset': 0., 'y_offset': 0,
                     'radius': 0.00125,
                     'wl': 500e-9, 'n_photons': 1000}
